# 🗳️ Blockchain Powered E-Voting System

## 📌 Project Overview

The **Blockchain Powered E-Voting System** is a secure, transparent, and decentralized voting platform built using blockchain technology. This system eliminates the limitations of traditional and centralized voting methods by ensuring that every vote is immutable, verifiable, and tamper-proof.

It allows users to cast their votes online through a web interface while maintaining voter privacy and ensuring election integrity.

---

## 🚀 Features

* 🔐 Secure voting using blockchain technology
* 🧾 Immutable and tamper-proof vote records
* 👤 One-person-one-vote authentication
* 🔎 Transparent and publicly verifiable results
* ⚡ Real-time vote counting
* 🌐 Decentralized system (no central authority)
* 🧑‍💻 User-friendly web interface

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/yourusername/blockchain-voting-system.git
cd blockchain-voting-system
```

### 2️⃣ Install Dependencies

```
npm install
```

### 3️⃣ Start Ganache

* Open Ganache
* Start a local blockchain

### 4️⃣ Compile Smart Contracts

```
truffle compile
```

### 5️⃣ Deploy Contracts

```
truffle migrate
```

### 6️⃣ Run the Application

```
npm run dev
```

## 🧪 How It Works

1. User connects using MetaMask wallet
2. Voter authentication is verified
3. A voting token is issued
4. User selects a candidate and casts vote
5. Vote is stored as a blockchain transaction
6. Results are updated in real-time
